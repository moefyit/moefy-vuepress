# Moefy VuePress

让你的 VuePress 变得更加可爱~

Warning

这些插件仅支持 VuePress V1 (opens new window)，不支持 VuePress V2 (opens new window) 及 VitePress (opens new window)。如果你想在非 VuePress V1 环境下使用 ribbon 或 cursor-effects 插件，可以尝试 moefy-canvas (opens new window)（事实上现在 ribbon 和 cursor-effects 插件只是 moefy-canvas 的一个简单包装）～
